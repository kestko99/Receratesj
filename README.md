# Rotemplate - Texture Copier Website

A 1:1 replica of the Rotemplate texture copier website for Roblox avatars.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dark/Light Mode Toggle**: Switch between themes with persistent preference storage
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Social Media Integration**: YouTube and Twitter social links
- **Bookmark-based System**: Instructions for the texture compilation tool

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with CSS variables for theming
- `script.js` - Dark mode toggle and interactive features

## How to Use

1. Open `index.html` in your web browser
2. Click the theme toggle button (top-right) to switch between light and dark modes
3. The "Compile Avatar" button is a draggable bookmark for the Roblox texture copier tool

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- SVG icons
- LocalStorage for theme persistence

## Design Highlights

- Large, bold typography for the main title
- Card-based layout for content sections
- Smooth hover effects and transitions
- Grid-based responsive layout
- Accessible color contrast in both themes
